## Usage

> Some usage instructions:
> 1) npm install -g webpack & npm install dependencies.
> 2) connect to mongo with your MONGO_URI (db-models/database.js; example: 'mongodb://localhost/steam') & npm run seed.
> 3) npm run build.
> 4) npm run start.
